# AYOKO NA
The Ayoko Na Rant App is a web-based platform designed for students and users who need a space to release their emotions anonymously.

#OVERVIEW
It allows users to post rants, interact with others’ posts, and receive support or guidance from the community and admin.
The app also includes admin controls for managing content, users, and system feedback


#FEATURES 
• Post Rants — Share your thoughts or frustrations anonymously.
• Admin Moderation — Admins can manage posts and users.
• User Authentication — Register, log in, and log out securely.

#TECHBACK
Frontend: HTML, CSS
Backend: PHP, MySQL (via Xampp)
Database: MySQL

#USER CREDENTIALS 
•  XAMPP (with Apache and MySQL)
• Browser (Chrome or Edge)
• VS Code or Sublime Text (or any PHP IDE)

#HOW TO RUN
Extract zip file and move to folder xampp/htdocs
Open xampp and click start on Apache and Mysql
Click phpmyadmin and import db_ayokona
Open browser and type localhost/ayokona/php/landing.php
